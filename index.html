Choreographies

  • distributed systems involve several peers
    Alice (a), Bob (b), and Charles (c)

  • one can have a local view on the system
    a = get phone number from b ; give phone number to c
    b = give phone number to a ; talk to c
    c = obtain phone number from a ; talk with b

  • one can have a global view on the system
    obtain phone [a,b] ; give information [a,c] ; talk [b,c]
